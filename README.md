Civic Connect is a web-based platform designed to help citizens report local civic issues such as:

Road damage
Garbage problems
Water leakage
Streetlight failures
Drainage issues

The application allows users to submit complaints with images and location details, helping authorities track and resolve issues efficiently.

Features
User-friendly complaint registration
Upload images related to civic issues
Store complaint details in database
Track submitted complaints
Simple web interface using HTML templates
Flask-based backend application
Technologies Used
Python
Flask
HTML
CSS
SQLite / Database
Bootstrap (if used)
